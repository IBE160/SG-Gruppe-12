## If Tests Fail:

### P0 Failures (Blocker):
- Fix immediately (< 30 min effort)
- Re-run tests
- Then proceed with closure

### P1 Failures (Minor):
- Document in TECH-DEBT.md
- Mark Epic 1 as done with known issues
- Fix in Sprint 2

### P2 Failures (Nice-to-have):
- Document only
- Defer to Sprint 2/3
